## 整理规则

整理目标是生成“来源摘录笔记”，供后续再分类、归入正式 `wiki/` 标签体系前使用。

本来源是国家药典委员会官网“《中国药典》执行专栏”，存在“栏目列表 → 文章/外链/附件”的层级。栏目文章数量较多，且不少文章正文较长或以 PDF、Word、Excel 附件承载，因此不把全部正文堆叠在主来源笔记中；主来源笔记保留入口、规则和摘录目录，具体内容按文章拆为分层摘录笔记。

1. 存放位置：
   - 主摘录笔记放入与来源笔记同名的文件夹：`source/CPC/CPC_《中国药典》执行专栏/`。
   - 明显不属于“化学药品制剂”主线，或仅为培训活动、预算、出版经销、会议新闻等低知识密度信息的条目，若为便于复核而保留处理痕迹，放入：`source/CPC/CPC_《中国药典》执行专栏_排除/`。

2. 增量整理：
   - 先通过接口读取栏目列表，再与主摘录目录和排除目录中的 `发布日期_文章标题.md` 比对。
   - 已存在同名摘录笔记时默认跳过，不覆盖；除非用户明确要求刷新或修正。
   - 新增文章按栏目发布日期与文章标题生成文件名：`发布日期_文章标题.md`。
   - 文件名需清理 Windows 不允许字符：`< > : " / \ | ? *` 等替换为安全字符，通常用 `、`，并删除标题内部空白以兼容标签/文件名习惯。

3. 筛选范围：
   - 本来源围绕“化学药品制剂”关键主题整理。
   - 优先摘录与化学药品制剂的药典执行、质量标准、注册标准衔接、上市后变更、制剂通则、无菌/微生物、制药用水、药用辅料、药包材、ICH 协调、药品标准管理等直接相关或可直接适用的内容。
   - 忽略仅与中药、中药饮片、中成药、医疗器械、体外诊断试剂、化妆品、保健品、纯生物制品/疫苗专题等相关的内容。
   - 若一篇文章或问答中混有多主题条目，应在条目/行级别筛选，只保留与本库主线直接相关或可直接适用的条目。
   - 旧培训通知、会议通知、预算、出版经销信息等即使出现在栏目列表中，通常不作为知识内容摘录；除非其正文包含可长期复用的技术要求或执行规则。

4. 摘录笔记 frontmatter：
   - 使用以下字段，frontmatter 与正文之间保留一个空行：
     ```yaml
     ---
     source_note: "[[CPC_《中国药典》执行专栏]]"
     source_url: "国家药典委员会文章页 URL"
     external_url: "外链 URL（如有）"
     article_title: "原文章标题"
     date: YYYY-MM-DD
     ---
     ```
   - `date` 使用栏目/文章接口中的发布日期。
   - `external_url` 仅在国家药典委员会文章为外链时填写。

5. 正文表格：
   - 正文整理为 3 列 Markdown 表格：`问题 / 解答 / 发布日期`。
   - 问答类文章尽量按问答拆行；非问答类文章以文章标题作为“问题”、正文或附件提取文本作为“解答”。
   - 表格单元格内换行用 `<br>`；正文中的 `|` 需转义为 `\|`；有效网页链接保留为 Markdown 链接。
   - 附件保留官网下载链接；PDF 附件可提取文本时，将提取文本并入对应摘录笔记，不能提取的 Word/Excel 等附件只保留链接，不臆造内容。

6. 抓取与异常处理：
   - 栏目列表接口：`https://www.chp.org.cn/three/sys/news/es/list?blockId=47&pageNo=1&pageSize=1000`。
   - 文章详情接口：`https://www.chp.org.cn/three/sys/news/es/detail?newsId=...`。
   - 附件下载接口：`https://www.chp.org.cn/three/anon/user/download?id=...&token=`。
   - 国家药监局 `nmpa.gov.cn` 等外链页面可能返回 HTTP 412 访问校验；遇到这类情况可生成占位摘录笔记，保留原文链接并说明未能直接提取正文，不臆造正文内容。
   - 抓取失败、正文为空、PDF 文本提取异常、或行级筛选不确定时，应在最终回复中提示人工复核。
