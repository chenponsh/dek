# Source Sync Records

本文件夹用于保存可纳入 git 的来源增量同步记录。

- 每次遍历/增量同步 source 笔记后，将结构化运行记录保存为 `source_sync_YYYYMMDD_HHMM_report.json`。
- 这里使用 `sync`，表示“将外部来源与本地 source 笔记进行增量对齐”：包括抓取远端、比对本地、写入新增、生成 rough 草稿、记录跳过/失败原因。
- 记录文件应包含：同步日期、逐来源状态、远端最新日期/数量、是否发现新增、是否生成 rough 草稿、跳过或失败原因。
- 临时抓取缓存、HTML、PDF、脚本调试输出仍放在 `_/`，不纳入 git；只有可复用的同步记录和方法说明放在本文件夹。
