## 新增问答

| 问题 | 解答 | 发布日期 |
| --- | --- | --- |
| 如何申请药品网络交易服务第三方平台备案? | 申请人可在上海市药品监督管理局官网（http://yjj.sh.gov.cn）点击“一网通办”，在搜索栏中输入“药品网络销售备案报告”，点击“药品网络交易第三方平台备案”，查看其办事指南（包含办理形式、需要资料、办理时限、证照形式、办理地址、法规设定依据、收费标准等信息）。 | 2023-06-13 |
| 药品上市许可持有人如何提交委托销售采购报告？ | 访问方式：申请人进入上海市药监局申报系统（https://xuke.yjj.sh.gov.cn）点击“一网通办登录”后，在新开页面依次点击【申报记录管理】（若显示）、【服务事项申报】、【药品生产流通质量管理备案和报告——上传入口】、【委托销售报告——在线申报】报告要求：如实填写委托方信息、受托方信息、委托销售的药品信息等，提交委托销售协议，并下载打印系统生成的申请报告，签章后上传。药监部门确认后的报告将通过一网通办推送到企业端。 | 2022-12-25 |
| 药品流通质量管理相关报告的提交方式？ | 申请人进入上海市药监局申报系统（https://xuke.yjj.sh.gov.cn）点击“一网通办登录”后，在新开页面依次点击【申报记录管理】（若显示）、【服务事项申报】、点击【药品生产流通质量管理备案和报告——上传入口】。选择项目后，填写简要说明和情况说明并上传报告材料。 | 2022-12-25 |
| 药品经营企业有没有向药品供应商等提供药品进销存数据的法定义务？ | 《药品管理法》规定，药品上市许可持有人、药品生产企业、药品经营企业和医疗机构应当建立并实施药品追溯制度，按照规定提供追溯信息，保证药品可追溯。国家药监局《关于药品信息化追溯体系建设的指导意见》规定，药品批发企业在采购药品时，向上游企业索取相关追溯信息，在药品验收时进行核对，并将核对信息反馈上游企业；在销售药品时，应向下游企业或医疗机构提供相关追溯信息。 | 2022-12-25 |
| 如何提交特殊情形所需要疫苗采购报告？ | "访问方式：申请人进入上海市药监局申报系统（https://xuke.yjj.sh.gov.cn）点击“一网通办登录”后，在新开页面依次点击【申报记录管理】（若显示）、【服务事项申报】、【药品生产流通质量管理备案和报告——上传入口】、【特殊情形所需要疫苗采购报告——在线申报】报告要求：如实填写疫苗采购信息，下载打印系统生成的申请报告，签章后上传。" | 2022-12-14 |
| 药品进口通关单如何办理？ | 根据《国家药监局关于启用新版药品和药材进口备案管理系统的公告》(2019年第107号)，国家药监局与海关总署依托国际贸易“单一窗口”，合作建设了药品和药材进口备案管理系统。报验单位登录http://www.singlewindow.cn 提交药品进口备案申请。待网上备案完成后，可至通关备案窗口打印纸质《进口药品通关单》。（窗口地址：上海市食品药品检验研究院，上海市浦东新区张衡路1500号，咨询电话：021-38839900） | 2022-12-14 |
| 如何申请医疗机构配制制剂许可？ | 申请人可在上海市药品监督管理局官网（http://yjj.sh.gov.cn）点击“一网通办”，在搜索栏中输入“医疗机构配制制剂许可”查看其办事指南（包含办理形式、需要资料、办理时限、证照形式、办理地址、法规设定依据、收费标准等信息）。 | 2022-12-14 |
| 什么是互联网药品信息服务资格证（告知承诺）？ | 选择告知承诺方式申请本许可，按要求提交材料并签署告知承诺文书后当场作出许可，我局将按相关规定组织后续监督检查，发现实际情况与承诺内容不符的，依法予以责令整改、撤销许可决定等处置，并录入企业诚信档案。 | 2022-12-14 |
| 如何查询申请事项的办理进度？ | 申请人可扫描受理通知书上的二维码查看办理进度，或登录上海市药品监督管理局政务网站（yjj.sh.gov.cn）“一网通办”中的“申报记录管理”栏目，通过统一审批编码查询。 | 2022-12-14 |
| 在提交申请后，收到被退回的短信，如何查看具体退回原因？是否需要从头开始填写申请？ | 申请人可登录上海市药品监督管理局政务网站（yjj.sh.gov.cn）“一网通办”中的“申报记录管理”栏目，在“申报管理”菜单中，根据统一审批编码查询相关申请项目的退回原因，并修改原先提交的资料，修改完成后再点击提交即可。 | 2022-12-14 |
| 已提交的申请想撤回修改，但找不到撤回选项。 | 申请人可登录上海市药品监督管理局政务网站（yjj.sh.gov.cn）“一网通办”中的“申报记录管理”栏目，在“申报管理”菜单中，根据统一审批编码查询相关申请项目。若该申请已被接收，则需向窗口提交撤回申请。 | 2022-12-14 |
